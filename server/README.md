# Call booking

This is the backend for the booking call Onja app

## Requirments
- node
- nmp
- docker 
- docker-compose 

## installation

- clone the project from the repo

### Development
- npm install
- npm run dev.start

### Production
- build your project using

```
$ docker-compose build
```


The API of the project will run under http://localhost:4000/graphql
